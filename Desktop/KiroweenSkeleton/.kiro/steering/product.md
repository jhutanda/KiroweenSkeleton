# Product Overview

KiroweenSkeleton is a full-stack dual-application productivity platform designed for Devpost hackathons. It combines a Smart Notes App with AI summarization and a Daily Task Manager with AI scheduling within a single shared codebase.

## Core Applications

- **Smart Notes App**: Note-taking with AI-powered summarization, version history, and search capabilities
- **Daily Task Manager**: Task management with AI scheduling, deadline notifications, and calendar timeline views

## Key Features

- Unified AWS Cognito authentication across both applications
- AI-powered productivity features (note summarization, task scheduling)
- Shared UI design system with light/dark theme support
- PWA capabilities with offline functionality
- Real-time data synchronization and browser notifications
- Export functionality (PDF for notes, JSON backup)
- Productivity analytics and scoring

## Target Use Case

Built for hackathon participants who need efficient note-taking and task management in a single, polished application that demonstrates modern full-stack development practices.
